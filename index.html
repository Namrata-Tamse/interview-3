1. Write a code to insert an image?


To insert an image in html, we use img tag.
In this tag we have to use src (source) attribute to provide the image url.
And other than that we give alt (alternate) attribute, whose value will be displayed
on the screen if the image url given in source is not correct.

Code:-

<h1>Image</h1>
<img src="link address of the image" alt="alternate text">

Output:-

Image

3.Create a table and explain how to achieve colspan?


In this example, the table includes a colspan in the third row of the tbody.
The first two rows of the table are part of the thead section, which contains the header
information for the table. The third row of the tbody contains the total monthly expenses for
rent and utilities, with the total amount spanning across both columns using the colspan attribute.
To achieve colspan in HTML, you can add the "colspan" attribute to a td or th element and specify the number of
columns that should be spanned. For example, if you want to span two columns, you would use colspan="2".
This allows you to merge cells in a table and create more complex table layouts.
